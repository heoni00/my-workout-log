# 내 운동일지

개인용 운동 기록 앱입니다. 운동 종목별 kg, 횟수, 세트와 음식 칼로리, 그날의 느낌을 날짜별로 저장합니다.

## 실행

`index.html`을 브라우저에서 열면 바로 사용할 수 있습니다. PWA 설치와 오프라인 캐시까지 확인하려면 로컬 서버로 실행하세요.

```bash
python3 -m http.server 5173
```

브라우저에서 `http://localhost:5173`으로 접속하면 됩니다.

## 저장 방식

기록은 브라우저의 `localStorage`에 저장됩니다. 다른 기기로 옮기려면 앱의 `요약` 탭에서 백업 내보내기/가져오기를 사용하세요.
