# 득근일지

귀여운 캐릭터가 있는 개인용 운동 기록 PWA입니다. 운동, 식단, 인바디, 운동 정보 메모를 날짜별로 저장합니다.

## 기능

- 홈: 오늘 날짜, 최근 운동 후 지난 일수, 최신 인바디, 캘린더
- 운동 일지: 시작/종료 시간, 운동명, kg, 횟수, 세트, 운동 메모
- 식단 일지: 아침/점심/저녁/간식/야식, 시간, 음식, 칼로리
- 내 몸 상태: 몸무게, 골격근량, 체지방량, 체지방률 히스토리
- 운동 정보 메모: 보조제, 운동 영상 URL, 루틴, 팁 저장

## 실행

`index.html`을 브라우저에서 열면 바로 사용할 수 있습니다. PWA 설치와 오프라인 캐시까지 확인하려면 로컬 서버로 실행하세요.

```bash
python3 -m http.server 5173
```

브라우저에서 `http://localhost:5173`으로 접속하면 됩니다.

## 저장 방식

기록은 브라우저의 `localStorage`에 저장됩니다. 다른 기기로 옮기려면 `메모` 탭의 백업 내보내기/가져오기를 사용하세요.
